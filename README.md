# クリスマス感謝祭2024 特設サイト & 参加申し込みフォーム

## 1. プロジェクト概要
郡山家庭教会で開催される「クリスマス感謝祭」のイベント告知および参加申し込みを行うための特設Webサイトです。
ポスターのデザイン世界観（青い夜空、金色の装飾）をWeb上で再現し、雪が降るアニメーションで季節感を演出しています。
バックエンド等の複雑な構築を行わず、GoogleフォームとGitHub Pagesを活用したサーバーレス構成で運用します。

## 2. イベント情報（仕様）
*   **イベント名**: クリスマス感謝祭
*   **日時**: 12月21日（日）
    *   一部（記念礼拝）: 10:00〜
    *   二部（感謝祭）: 11:00〜
*   **場所**: 郡山家庭教会
*   **主なコンテンツ**:
    *   ステージ（聖歌隊、ダンス、バンド）
    *   ブース（焼鳥、チヂミ、おにぎり、ナゲット、ケーキ、パネル展）
    *   シールラリー企画

## 3. システム構成
メンテナンスコストを最小限に抑えるため、以下の構成を採用しています。

*   **フロントエンド**: HTML5, CSS3, JavaScript (Vanilla JS)
*   **バックエンド/DB**: Google Forms (iframe埋め込み)
*   **インフラ/ホスティング**: GitHub Pages
*   **バージョン管理**: Git / GitHub

### アーキテクチャ図
`[User]` <--> `[GitHub Pages (HTML/CSS/JS)]`
                      |
                      v
             `[Google Forms (Data Entry)]`

## 4. 機能要件
### 4.1. 表示機能
*   **レスポンシブデザイン**: スマートフォン（縦長）とPC（横長）の両方で崩れずに表示されること。
*   **情報掲載**: ポスター記載のイベント詳細、ブース案内、アクセスマップをテキストと画像で表示。

### 4.2. アニメーション機能 (JavaScript)
*   **降雪エフェクト**:
    *   画面上部からランダムな位置・サイズで「雪の要素」を生成。
    *   下方向へアニメーション移動し、画面外に出たら要素を削除（メモリ管理）。

### 4.3. 申し込み機能
*   **Googleフォーム連携**:
    *   サイト内に申し込みフォームを埋め込み表示。
    *   ページ遷移なしで申し込み完了まで行えること。
    *   収集項目：氏名、参加人数、連絡先（メールアドレス等）。
